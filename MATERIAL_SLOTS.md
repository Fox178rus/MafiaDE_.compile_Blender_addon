# Material Slot Mapping (Manual)

## Что такое slot

Slot — это **глобальный ID материала в движке Illusion Engine**.
Он не является индексом в каком-либо XML файле или хешем имени.
Назначается при компиляции материалов и хранится только в скомпилированных данных.

Один slot может использоваться несколькими зданиями (например slot=2 — базовая штукатурка).

## Вывод

Slot → имя текстуры: связь не автоматизирована из распакованных файлов.
Slot — глобальный ID движка, хранится только в Blueprint/скомпилированных данных.

Для практического использования: подбирать текстуры по имени здания из базы MAFIA_DE_TEXTURES.

## Текстуры в базе

Все нужные текстуры **есть** в `MAFIA_DE_TEXTURES\LOD0\D\`:
- `lh_02_house_a_v1---d.dds` ✓
- `lh_02_house_a_v1_keyed---d.dds` ✓
- `lh_uni_wall_bricks_a_v1_dirty---d.dds` ✓ (исправлено)

Текстуры зданий перечислены в lod1 SDS XML (SDSContent.xml).
Все они были распакованы и перемещены в базу при сборке MAFIA_DE_TEXTURES.

## Рабочий метод

Для каждого здания:
1. Берём имя типа из nomesh (например `lh_02_house_a_v3`)
2. Убираем суффикс версии: `lh_02_house_a`
3. Ищем в LOD0/D все файлы с этим префиксом
4. Назначаем текстуры мешам вручную по slot маппингу
